# Общие сведения

## Назначение

RFID-считыватель предназначен для бесконтактного считывания уникальных идентификаторов RFID-меток и карт стандарта EM4100 на частоте 125 кГц. Считанный идентификатор передаётся по интерфейсу UART на подключённый компьютер.

Область применения: учебные и исследовательские задачи, системы контроля доступа.
